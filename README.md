# TIC_IPG203_S25_Project
تم تقسيم المشروع إلى ملفات منفصلة باستخدام خاصية Partial Classes لتعزيز مبدأ التعاون وتوزيع المهام:  

IPlayable.cs: تعريف الواجهة (Interface) الأساسية للنشاطات.  

MediaContentProperties.cs: يحتوي على الحقول والخصائص المحمية (Encapsulation).  

ContentModels.cs: يحتوي على الأصناف المشتقة (Inheritance) مثل الفيديو والكتب والألعاب.  

ContentManager.cs: المسؤول عن إدارة المحتوى وإطلاق الأحداث (Events & Delegates).  

Program.cs: نقطة انطلاق البرنامج وتجربة الأكواد.
